About this project

The Idea

American football is a game characterized by high demands for strategy. From the players on the field to specific characteristics that change frequently throughout the game, it can be challenging to quantify the value of specific plays and actions within a play. This project focuses on two main lines. Firstly, analyzing the multiple data sources collected on football and offering an overview of what the data on teams from the 1960s to today is saying. Then, combining the collected data with Machine Learning algorithms and obtaining a prediction for the matches of the regular season 2022-2023.

Data Source

The datasets come from two sources. The first is the Sportsreference dataset. It contains the main outcomes of interest (yards gained on the play and play type) and contains rich features related to the situation of the game that may give some predictive power to what will happen in the play. For example, the features include the current score of the game and the yard line that the offensive team is on. The second source is FiveThirtyEight which contains forecasts based on 50,000 simulations of the season and updates after every game.

The Structure

The project structure is divided into the back-end and front-end. As for the back-end work, Pandas, Matplotlib and Seaborn were used for data analysis; for prediction, Sklearn library with SVM and Adaboost algorithms. For the front-end work, a Flask application was created using HTLM and CSS; dashboards were built with Tableau to visualize the data.
